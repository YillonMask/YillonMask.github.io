## 想法

1. **同一天两场 agent 会议覆盖两个垂直** — Interrupt 在 SF 谈 enterprise agent infra (Clay/Rippling/Workday)，NXT BLD 在伦敦谈 AEC agent (Forma/Hypar/Motif)。两个领域用的 mental model 越来越像：cloud-native authoring + agentic layer + 行业数据 schema。今晚或者明早值得看 Interrupt 主题演讲录像，特别是 Harrison + Jensen 那场。
2. **Anthropic 的"vertical packaged"打法是这一波 platform 公司演化的必然** — Claude for Legal / SMB 不是单纯卖 API，是卖"已经写好工作流的环境"。意味着接下来 12 个月，platform 厂会和 vertical SaaS（Harvey / Clio / DocuSign）出现既合作又抢饭吃的拉扯。Claude 选了"做 Cowork 环境 + 集成 SaaS"，OpenAI 走的是 Codex Chrome / 工作流 plug 的路径。
3. **"AI 预测健康事件"是一个长期 narrative** — Oura 公开提到要做"提前几年预测心梗 / 中风"，FDA 同步在血压认证。如果这条产品线跑通，wearable 行业的估值锚点从 "DTC 硬件 + 订阅" 切到 "predictive medicine 平台"，倍数完全不同。值得跟一下 Oura 和 Apple 后续在 FDA cleared 上的节奏。
4. **本地 agent 的硬件门槛在变** — Show HN 出现 26M 参数模型也能做 tool calling，叠加 dual Blackwell + 128GB Mac 的本地 agent 热潮，意味着"端侧 agent stack"会在 6–12 个月内出现真的可用产品。做 dev tooling 的应该重视。
