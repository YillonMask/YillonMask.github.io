想法

  1. LangChain 在 Interrupt Day 2 这波"全栈 agent infra"出货是教科书时刻 — 他们把"做 agent 公司会自己造的 10 件事"打包成 LangSmith Engine + Deep Agents + SmithDB + Sandboxes + Gateway，意思是"你别自己造了，按月付我"。和 Anthropic 6/15 agent 计费、Cursor agent dev environments 一起看，是同一条主线：agent 时代第一波 platform layer 在固化。下一年的赢家不在模型，在 runtime / observability / sandbox / governance 这一层。
  2. Xpanner 这个 case 把"AI × 物理基建"的 PMF 跑出来了 — 不是"用 AI 优化设计图"，是直接把 AI 自治系统装到现成机械上，主战场还选了"AI 数据中心 + 太阳能 + 储能"这三个 capex 暴涨的下游。19/20 的渗透率说明：施工业主真的愿意为 ROI 清楚的 automation 付订阅费。CE 背景做 AI × 建造的话，"给现有机械加 AI"比"做全新 modeling tool"近期 ROI 高得多。
  3. Wearables 的"医疗化"vs"OS-化"分叉今天明确 — Whoop + Abbott（医疗器械集团入股）、Oura + Dexcom（CGM 巨头入股）走的是 medical device 路；Google Health 5/19 上线、Apple Health 持续推进，把 sleep / readiness / AI coaching 标准化进 OS。中间没有第三条路 —— 没有真人医生 / EHR / FDA 路径的小品牌会被 OS 端 commoditize。
  4. AI 周活的真实意义在 Codex 移动端 — Codex 一个多月从 3M → 4M+ 周活；OpenAI 把它搬上手机，配合"长任务远程跑"的 UX，等于把 agent 引入 ambient computing。Anthropic 把 Claude 接进 Office 365 也是同一逻辑：agent 的下一战场是"你不在键盘前的时间"。

值得深入研究

  • LangSmith Engine 的 production failure clustering / root cause 拆解机制 — 直接对标 Honeycomb / Datadog 的 agent 版本，技术细节值得看。
  • Xpanner Automation-as-a-Service 的硬件改装 SOP — 决定能不能在中国 / 东南亚 / 中东复用。
  • Abbott × Whoop 后续 product roadmap — 会不会出现真的 "wearable + CGM + 真人医生 + 处方" 端到端产品。
  • Google Health app 5/19 上线后第三方 wearable 的接入条款 — 决定 Oura / Whoop 是被吸收还是另起平台。
  • Fast-Slow Training 是不是能跑通 small model 蒸馏；Model-First Reasoning 在 enterprise planning 工具里的可用性。
