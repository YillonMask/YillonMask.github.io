想法

  1. Karpathy 加入 Anthropic 是这一波 AI 人才 / 资本 / 算力三重集中的最新一个 data point — 配合 Q1 2026 四家公司吸走 65% VC 资金，行业已经进入"赢家通吃"的物理形态：你想做 frontier，必须同时有顶级人才 + 大额资本 + 几百兆瓦算力。剩下的赛道——agent infra、coder 工具、行业 vertical、wearable 平台——会变成下一层博弈，但 frontier model 这一层基本封顶了。
  2. Cursor Composer 2.5 用 Kimi K2.5 当底座是产业链信号 — 第一次有西方头部 AI 编码工具公开把中国开源 LLM 作为商业产品的训练基础。意味着：(a) 中国开源 LLM 的国际商用 license / 安全审查问题被市场接受度上行；(b) "coder 模型用专用底座 + synthetic 训练"成为新范式，frontier 通用模型在编码这一档不再是必须；(c) Anthropic / OpenAI 接下来要决定要不要走同方向，否则会被 unit economics 压死。这是开源 LLM 真正第一次撼动闭源商业的标志事件。
  3. AECOM 在全球 AEC AI 上的动作明显加速 — 5/18 收 Consigli $390M，5/19 出现在印度 TechTalks 主台。这是顶级工程顾问把 AI 当 M&A + 全球渠道双重战略，不是局部试点。CE 背景做 AI × 建筑的人，今天最重要的判断是：未来 12 个月，AEC × AI 的赢家会是"独立 startup"还是"被顶级顾问吸纳的内部团队"？我现在更倾向后者——理由是 AEC 行业付费意愿真正集中在顶级工程顾问手里，不在 IT 部门。
  4. Wearable 监管化是这一周第二条暗线 — 美国 HIPAA 立法提议 + Google Health 5/19 上线 + Whoop $575M G 轮发声同步发生。如果 wearable 数据进入 HIPAA 边界，Whoop / Oura 的"临床路线"反而是利好（他们已经在和真人医生 + EHR 对接），Fitbit Air 这种纯消费产品反而面临成本激增。短期监管不确定性，会强化"医疗 wearable + 消费 wearable"的物理分裂。
  5. 今晚 Nvidia 财报后市场情绪定调三件事：(a) AI 算力 demand 是真稀缺还是预期透支；(b) Anthropic Code w/ Claude / Cursor Composer 2.5 / OpenAI Codex Dell 这些 platform 故事能不能在估值上撑住；(c) Armada / Iceotope 这种数据中心物理基础设施 startup 的下一波估值。重点不是 Q1 那个 $78B beat 多少，是 Q2 那个 $86B 共识能不能被显著抬高。真正能压住市场的不是 miss，是 in-line 但 guide 不及预期。
