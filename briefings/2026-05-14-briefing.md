## 想法

1. **Anthropic 这次"agent 单独计价"是 AI 商业模式的关键节点** — 把"你跟我聊"和"你的 agent 替你做事"在定价上区分开，意味着 AI 公司开始按"工作量 × 自动化程度"收费，而不是按"订阅人头"。这套定价逻辑铺开之后，所有做 wrapper / agent 应用的 startup 都要重新算 unit economics。最痛的是 Pro 档（$20 chat + $20 agent 顶天 = $40），最有利的是 Max 20x 这种 power user 档。值得跟下 OpenAI 怎么响应（"OpenAI courts agent users"已经在新闻里）。
2. **Dexcom + Oura 是 wearable → 医疗器械的标志事件** — CGM 一直是医疗器械序列里离 FDA 最近的品类，smart ring 是 DTC 消费序列里数据最丰富的。两者数据打通 + Dexcom 主动入股，等于把"消费传感 → 医疗诊断 → 处方建议"这条链路第一次按现金流接通。Apple、Whoop、Samsung 接下来必须给出自己的 CGM/EHR/医师对接版本。
3. **NXT Activate 是 AEC × AI 元年的 institutional 信号** — Bentley 出钱，AEC Magazine 出渠道，$3M / 3 年虽然不大，但意味着 AEC AI startup 第一次有专门的 funnel。这跟 a16z 当年做 Crypto Startup School 是同一个剧本，只是产业链晚 8 年。**对 CE 背景想做 AI × 建筑的人来说，这是值得追踪的入口。**
4. **AI capital 集中在 infra（Anduril $5B、Fractile $220M、Mind Robotics $400M）** — 5/13 一天三个 9 位数往上的 deal 全在 infra / 国防 / 机器人，应用层只看到 SMB / 健康 / 合规这种 vertical SaaS。"AI 是真的 frontier 还是只是 SaaS 加点 LLM" 的分野在变清楚，钱在投硬件 / 算力 / 物理世界 actuator。
