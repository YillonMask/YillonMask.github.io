想法

  1. 本地 agent 硬件军备的"中端空白" — LocalLLaMA 的对话已经从"模型怎么选"变成"5090 还是 M5 Max"。这意味着至少有一批 power user 在认真用本地 agent 替代云服务，而 12–16GB GPU 的"够用 agent stack"是空白。做 dev tooling / agent runtime 的人可以认真想这个区间。
  2. AEC × AI 进入临界点 — Autodesk 公开喊 agentic BIM + NXT BLD 阵容 + 工地 CV 三巨头都 GA，意味着行业要在 12–24 个月内分叉成"设计端 / 施工端 / 运维端"三条独立的 AI 路径。哪一端先出真正 PMF 值得跟踪（个人感觉施工端最容易，因为 ROI 最硬）。
  3. Wearables 护城河从"算法"转向"医疗体系绑定" — Whoop 接真人医生 + EHR，Oura 接处方，本质都是把消费电子搬进医疗服务链。$99 Fitbit Air 不构成正面威胁（目标人群不同），但加速了行业向 "health outcome 平台"的演化。wearable + agent + EHR 的端到端组合是值得自己 sketch 一下的方向。
  4. Compute 已经 commoditize 不动了 — Anthropic 一周内绑了 5 个 compute 供应商；frontier model 的成本继续往下走，但企业数据 + 集成才是护城河（看 Anthropic-MS365、OpenAI-Codex-Chrome、Cursor-Teams 的路径就懂）。做产品的应该把"接进哪个企业 surface"当作主要战略问题，而不是"模型选哪个"。
